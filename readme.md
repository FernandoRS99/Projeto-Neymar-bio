# 🔗 LinkBio Neymar Jr

Projeto desenvolvido como exercício prático durante meus estudos de **Front-End**.

A proposta foi recriar uma página estilo **LinkBio**, utilizando apenas **HTML e CSS**, transformando um layout proposto em uma interface funcional.

## 🌐 Visualizar o projeto

Acesse o projeto online:

👉 https://fernandors99.github.io/Projeto-Neymar-bio/

## 📚 Conceitos praticados

Durante o desenvolvimento foram aplicados alguns conceitos importantes de desenvolvimento web:

- Estrutura semântica com **HTML**
- Estilização e layout com **CSS**
- Alinhamento utilizando **Flexbox**
- Uso de **gradientes**
- Pseudo-elementos (`::before`)
- Vídeo como plano de fundo
- Construção de **tabelas de dados**

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- Visual Studio Code
- Git & GitHub

## 🎯 Objetivo do projeto

Este projeto faz parte da minha jornada de aprendizado em **desenvolvimento web**, com foco em compreender como transformar layouts em páginas funcionais utilizando código.

Mais do que apenas reproduzir o design, o objetivo foi entender como cada elemento da interface é construído e estilizado.
